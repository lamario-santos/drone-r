# Hardware Resources Guide for Flipper Zero and Drone Integration

## 1. Introduction
Flipper Zero is a portable multi-tool for geeking, hacking, and integration with devices. This document outlines the integration of Flipper Zero with various drone models.

## 2. Required Hardware
- **Flipper Zero**: Versatile device for interfacing with drones.
- **Supported Drones**: 
  - DJI Mini 2
  - Parrot Anafi
- **Additional Sensors**: GPS module, LiDAR, cameras as optional sensors.

## 4. Configuration Steps
1. Install required software on Flipper Zero.
2. Connect Flipper Zero to the drone using the specified pins.
3. Calibrate sensors as needed.

## 5. Testing and Troubleshooting
- **Issue**: Flipper Zero not recognizing the drone
  **Solution**: Ensure connections are secure.
- **Issue**: Sensors not reporting data
  **Solution**: Verify sensor configurations.
